Detailed Description

Stores an exchange

buy_orders keeps a list of th buy orders. Similarly, sell_orders keeps a list of sell orders.

ask_prices maintains a count of the number of elements available at the given ask_price. bid_prices maintains a count of the number of elements available at the given bid_price
